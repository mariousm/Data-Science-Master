PEC2: Redes Neuronales Convolucionales con KERAS¶

A lo largo de esta prctica vamos a implementar varios modelos de redes neuronales para clasificar las imgenes de la base de datos CIFAR10.

La base de datos CIFAR10 est formada por imgenes en color de dimensiones 32x32 pxels que pertenecen a 10 clases:

  • airplane (label 0)
  • automobile (label 1)
  • bird (label 2)
  • cat (label 3)
  • deer (label 4)
  • dog (label 5)
  • frog (label 6)
  • horse (label 7)
  • ship (label 8)
  • truck (label 9)

Esta base de datos est compuesta de 60.000 registros que se dividen a su vez en 2 conjuntos: 50.000 imgenes para el conjunto de entrenamiento y 10.000 imgenes para el conjunto de test.

En el ejercicio llevaremos a cabo la tarea de construir varios modelos con el objetivo de conseguir la mayor precisin al realizar la clasificacin de las imgenes. Para ello implementaremos y analizaremos los siguientes modelos:

  1. Una red neuronal artificial completamente conectada
  2. Una red neuronal convolucional de una capa
  3. Una red neuronal convolucional profunda

En todos los casos se utilizar la librera Keras con Tensorflow para la implementacin del modelo, la compilacin y el entrenamiento.

Finalmente, realizaremos una prediccin de las imgenes con el mejor modelo implementado.
